# Bakery Shop Website

## Overview

The Chips Shop Website is a modern and responsive website designed to reveal the products of a shop located in Egypt. The website include about section that insures the good quality of chips, contact information including address, phone, mail, and social media link. The website also includes the menu with stunning UI and smothly animation.

## Features

Responsive design for desktop, tablet, and mobile devices.
Home page with chips products.
Chips menu displaying different types of breads with their prices.
About Us section introducing the ethics of the shop, and the quality of chips.
Contact section with adress, phone number, email, and social media links.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Usage
Developed as a chips shop website project by enhancing the marketing startegy of the shop.
